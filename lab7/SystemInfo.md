# System Information of the Virtual Machine

## Processor, RAM, and Network Information

### Tool: `lscpu`, `free`, `ip`

**Processor Information**:
- **Tool**: `lscpu`
- **Command**: `lscpu`

**RAM Information**:
- **Tool**: `free`
- **Command**: `free -h`

**Network Information**:
- **Tool**: `ip`
- **Command**: `ip a`

## Operating System Specifications

### Tool: `lsb_release`

- **Command to display OS specifications**: `lsb_release -a`
